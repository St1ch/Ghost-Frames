# Ghost Frames

Makes item frames invisible so players can click chests through frames, and provides a GUI to control frame settings.

## Features
- Invisible frames: only the item is visible
- Click-through to containers behind invisible frames
- Shift + Right Click GUI to toggle visibility and item rotation lock
- Frame settings saved in PersistentDataContainer

## Compatibility
- Minecraft: 1.18–1.21.x
- Cores: Bukkit, Spigot, Paper, Folia

## Installation
1. Build the plugin or use the jar from build/libs
2. Place the jar in the plugins folder
3. Restart the server

## Usage
- Shift + Right Click a frame to open the GUI
- Right Click an invisible frame to open the container behind it

## Build
```bash
.\gradlew.bat build
```
